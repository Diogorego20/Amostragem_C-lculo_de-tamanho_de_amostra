1. Introdução

Estas resoluções apresentam as soluções dos exercícios da Lista de Cálculo de Tamanho de Amostra e da Seção 8 do livro "Amostragem com R" (Amostragem Sistemática), integrando a teoria apresentada nas Aulas 08 e 09 da disciplina Amostragem I.

A amostragem sistemática (AS) é um método que oferece vantagens significativas em termos de precisão e facilidade de implementação quando comparada à amostragem aleatória simples (AAS), especialmente quando a população possui uma ordenação natural que captura a variabilidade do fenômeno.

2. Amostragem Sistemática (Seção 8)

Exercício 8.4: Comparação de Eficiência (AS vs AAS)

Enunciado: Tabela 8.5 mostra o número de sementes germinadas em 200 canteiros. Calcule a variância do estimador da média para AS e AAS (n=10).

Solução:

Parâmetros:

  • N = 200 (tamanho da população)
  • n = 10 (tamanho da amostra)
  • K = 20 (intervalo de seleção)
  • a = 20 (número de amostras sistemáticas possíveis)

a) Variância sob Amostragem Sistemática:

Var(ȳ_AS) = (1/a) × Σ(ȳ_r - Ȳ)²
Var(ȳ_AS) = 0.8949

b) Variância sob Amostragem Aleatória Simples:

Var(ȳ_AAS) = (1 - n/N) × S² / n
Var(ȳ_AAS) = (1 - 10/200) × S² / 10
Var(ȳ_AAS) = 11.001

c) Análise de Eficiência:

EPA (Eficiência Relativa) = Var(AS) / Var(AAS) = 0.8949 / 11.001 = 0.0813

A Amostragem Sistemática é aproximadamente 12.3 vezes mais precisa que a AAS neste caso.

Conclusão: A AS é significativamente mais eficiente que a AAS. Isso ocorre porque a população foi ordenada de forma que as colunas da tabela criam uma estratificação implícita. A AS, ao selecionar uma unidade de cada coluna, garante que a variabilidade entre as colunas seja bem representada em cada amostra.

Referência: Aula 09, Slide 2 - Variância do Estimador da Média na AS

Exercício 8.5: Ocupantes de Veículos (Intervalo de Confiança)

Enunciado: N=105, K=10, r=8. Dados: 3, 5, 7, 2, 3, 4, 5, 4, 4, 2. Estime a média e construa um IC 95%.

Solução:

a) Plano Amostral:

Amostragem Sistemática Simples com N=105, K=10, n=10

b) Veículos Selecionados (números de ordem):

8, 18, 28, 38, 48, 58, 68, 78, 88, 98

c) Estimativa da Média:

ȳ = (3 + 5 + 7 + 2 + 3 + 4 + 5 + 4 + 4 + 2) / 10 = 3.9 ocupantes

d) Intervalo de Confiança 95%:

Suposição: População em ordem aleatória (Solução 1 - Aula 09, Slide 13)

V̂₁ᴬˢ = (1/n - 1/N) × s²
V̂₁ᴬˢ = (1/10 - 1/105) × 2.3222 = 0.2101
EP = √0.2101 = 0.4583
IC 95% = 3.9 ± 1.96 × 0.4583 = [3.002, 4.798]
Resposta: A média estimada é 3.9 ocupantes por veículo, com intervalo de confiança de 95% sendo [3.002, 4.798].

Referência: Aula 09, Slide 13 - Estimador da Variância (Solução 1)

3. Cálculo de Tamanho de Amostra

Questão 01: Tamanho de Amostra para Proporções

Enunciado: De acordo com o US Bureau of Labor Statistics, 71% dos trabalhadores tinham acesso a planos de assistência médica. Calcule o tamanho de amostra para estimar essa proporção usando diferentes critérios de precisão.

Solução:

a) Coeficiente de Variação (CV) = 10%:

n = (z² × (1-P)) / (CV² × P)
n = (1.96² × (1-0.71)) / (0.10² × 0.71)
n = (3.8416 × 0.29) / 0.0071 = 1,142

b) Erro Padrão (EP) = 3 pontos percentuais = 0.03:

n = (P × (1-P)) / EP²
n = (0.71 × 0.29) / 0.03²
n = 0.2059 / 0.0009 = 229

c) Margem de Erro (MOE) = 3 pontos percentuais = 0.03:

n = (z² × P × (1-P)) / MOE²
n = (1.96² × 0.71 × 0.29) / 0.03²
n = (3.8416 × 0.2059) / 0.0009 = 880

d) Semi-larguras dos Intervalos de Confiança 95%:

Critério Tamanho (n) Semi-largura (z × EP)
CV = 10% 1,142 0.0071 (0.71%)
EP = 3% 229 0.0588 (5.88%)
MOE = 3% 880 0.0300 (3.00%)
Conclusão: O tamanho de amostra varia significativamente conforme o critério de precisão escolhido. O CV resulta em maior tamanho amostral (n=1,142) porque é mais restritivo, enquanto o EP resulta no menor (n=229).

Questão 02: Prevalência de Característica Rara

Enunciado: Um pesquisador deseja estimar a prevalência de uma característica rara (P=2%) com margem de erro (MOE) = 0.005.

Solução:

a) Tamanho Amostral Necessário:

n = (z² × P × (1-P)) / MOE²
n = (1.96² × 0.02 × 0.98) / 0.005²
n = (3.8416 × 0.0196) / 0.000025
n = 3,006

b) Análise de Sensibilidade:

Como o pesquisador está incerto sobre a prevalência real, é recomendado calcular n para diferentes valores de P:

Proporção (P) Tamanho (n) Observação
P = 0.01 (1%) 1,530 Menor que estimado
P = 0.02 (2%) 3,006 Estimado pelo pesquisador
P = 0.05 (5%) 7,299 Maior que estimado
P = 0.50 (50%) 153,664 Pior caso (máxima variância)
Recomendação: Se o pesquisador está muito incerto, usar P=0.50 (pior caso) resultaria em n=153,664. Uma abordagem mais prática seria usar P=0.02 (n=3,006) ou realizar um estudo piloto para refinar a estimativa.

Questão 03: Detectar Mudança na Renda Média

Enunciado: A renda disponível média era $6,100 em 2002. Deseja-se detectar um aumento para $7,000 em 2010. Variância relativa unitária = 2.5, α = 0.05, poder = 0.80.

Solução:

Parâmetros:

  • μ₀ = $6,100 (média em 2002)
  • μ₁ = $7,000 (média esperada em 2010)
  • Δμ = $900 (mudança a detectar)
  • Variância relativa unitária = 2.5
  • S = √(2.5 × 6,100²) = $9,681
  • α = 0.05 (nível de significância bilateral)
  • Poder = 0.80 (β = 0.20)
z_α/2 = 1.96 (para α = 0.05 bilateral)
z_β = 0.84 (para poder = 0.80)
n = ((z_α/2 + z_β)² × S²) / (μ₁ - μ₀)²
n = ((1.96 + 0.84)² × 9,681²) / 900²
n = (7.84 × 93,722,761) / 810,000
n = 905
Resposta: Um tamanho de amostra de n = 905 é necessário para detectar um aumento de $900 na renda média com 80% de poder e 5% de nível de significância.

Questão 04: Tamanho de Amostra para Estudo de Obesidade em Crianças

Enunciado: Determinar tamanho amostral para detectar mudanças no IMC de crianças (6-7 anos) com poder = 80% e α = 0.05.

Solução:

a) Detectar aumento de 1.5% em IMC (μ₀ = 17.5):

μ₁ = 17.5 × 1.015 = 17.7625
Δμ = 0.2625
S = 0.70
n = ((1.96 + 0.84)² × 0.70²) / 0.2625²
n = (7.84 × 0.49) / 0.0689
n = 56

b) Detectar aumento de 3.0% em IMC:

μ₁ = 17.5 × 1.03 = 18.025
Δμ = 0.525
n = ((1.96 + 0.84)² × 0.70²) / 0.525²
n = 3.84 × 0.49 / 0.2756
n = 14

c) Detectar aumento de 0.5% em IMC:

μ₁ = 17.5 × 1.005 = 17.5875
Δμ = 0.0875
n = ((1.96 + 0.84)² × 0.70²) / 0.0875²
n = 3.84 × 0.49 / 0.00766
n = 246
Mudança Detectada Δμ (IMC) Tamanho (n)
1.5% 0.2625 56
3.0% 0.525 14
0.5% 0.0875 246
d) Comentário: Existe uma relação inversa entre o tamanho da mudança a detectar e o tamanho amostral necessário. Quanto menor a mudança que se deseja detectar, maior deve ser a amostra. Neste caso, detectar uma mudança de 0.5% requer 246 pessoas, enquanto detectar 3.0% requer apenas 14.

Questão 05: Estudo Longitudinal com Atrito

Enunciado: Acompanhamento de desemprego em dois períodos. Taxa de desemprego T1 = 8%, esperada queda de 1.5 p.p. em T2. Cooperação esperada = 75%. Poder = 80%, α = 0.05 (unilateral).

Solução:

a) Tamanho de Amostra Necessário:

Parâmetros:

  • p₁ = 0.08 (taxa de desemprego em T1)
  • p₂ = 0.065 (taxa esperada em T2, queda de 1.5 p.p.)
  • Δp = 0.015 (mudança a detectar)
  • Teste unilateral: z_α = 1.645
  • Poder = 0.80: z_β = 0.84
  • Taxa de cooperação = 75%
p̄ = (p₁ + p₂) / 2 = (0.08 + 0.065) / 2 = 0.0725
n = ((z_α + z_β)² × [p₁(1-p₁) + p₂(1-p₂)]) / Δp²
n = ((1.645 + 0.84)² × [0.08×0.92 + 0.065×0.935]) / 0.015²
n = (6.2001 × [0.0736 + 0.0608]) / 0.000225
n = (6.2001 × 0.1344) / 0.000225
n = 3,695

Ajuste para Atrito:

Como apenas 75% da amostra T1 cooperará em T2, é necessário aumentar a amostra inicial:

n_ajustado = n / taxa_cooperação
n_ajustado = 3,695 / 0.75 = 4,927
Resposta (a): Tamanho de amostra necessário em T1 = 4,927 pessoas. Esperando 75% de cooperação, teremos 3,695 pessoas em T2, suficientes para detectar a queda de 1.5 p.p. com 80% de poder.

b) Poder com Amostra de 500 Pessoas:

Com n = 500 (T1) e 75% de cooperação = 375 (T2)
z_calc = (√375 × 0.015 - 1.645 × √[0.0725×0.9275]) / √[0.08×0.92 + 0.065×0.935]
z_calc = (19.36 × 0.015 - 1.645 × 0.2596) / 0.3668
z_calc = (0.2904 - 0.4270) / 0.3668 = -0.373
Poder = Φ(-0.373) ≈ 0.35 (35%)
Resposta (b): Com uma amostra de 500 pessoas, o poder para detectar uma mudança de 1.5 p.p. é apenas 35%, muito abaixo do desejado (80%). Portanto, a amostra de 500 é insuficiente para este estudo.

4. Conclusão

Este relatório apresentou as resoluções completas dos exercícios de Amostragem Sistemática (Seção 8) e Cálculo de Tamanho de Amostra, integrando a teoria estatística com aplicações práticas.

Os resultados demonstram que:

  • A Amostragem Sistemática pode ser significativamente mais eficiente que a AAS quando a população possui uma ordenação favorável;
  • O tamanho de amostra varia substancialmente conforme o critério de precisão escolhido;
  • A relação entre tamanho de mudança a detectar e tamanho amostral é inversa;
  • Estudos longitudinais com atrito requerem amostras maiores para compensar a perda de respondentes.
